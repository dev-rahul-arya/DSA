# QUEUE DATA STRUCTURE

- Stack: LIFO discipline
- Queue: FIFO discipline

## QUEUE ADT

### DATA
- Storage
- Insertion at End
- Deletion at End

### METHODS
- Enqueue
- Dequeue
- FirstVal
- LastVal
- Peek(pos)
- IsEmpty
- IsFull

__Note:__   1. Queue is not limited to counters
            2. Queue can be implemented by
                - Arrays
                - Linked List
                - Other ADTs
