# DOUBLE ENDED QUEUE 

Queue
- Insertion from Rear.
- Deletion from Front.

DEQueue
- Insertion from Rear + Front.
- Deletion from Rear + Front.
- Two Types:
    - Restricted Input => Insertion from Front not allowed
    - Restricted Output => Deletion from Rear not allowed

## QUEUE ADT
1. Data -> Same as Queue
2. Operations -> enqueueF(), dequeueF(), enqueueR(), dequeueR()
